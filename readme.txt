Это веб-приложение для просмотра информации о фильмах. В приложении можно найти информацию о популярных фильмах, использовать поиск по фильмам и просматривать подробную информацию о каждом фильме.

## Основные функции

1. **Главная страница (Homepage):**
   - Отображение списка популярных фильмов с возможностью сортировки и фильтрации.
   - Добавление фильмов в избранное и список "Посмотреть позже".

2. **Страница поиска (SearchPage):**
   - Поиск фильмов по названию и фильтрация по жанрам.

3. **Страница с подробной информацией о фильме (MovieDetailsPage):**
   - Отображение подробной информации о фильме, включая название, описание, рейтинг, актеров и комментарии.
   - Возможность добавления фильма в избранное и список "Посмотреть позже".
   - Добавление комментариев к фильму.

## Используемые технологии

- React: для создания пользовательского интерфейса.
- React Router: для навигации между страницами.
- Axios: для работы с API The Movie Database (TMDb).
- CSS: для стилизации компонентов.
- FontAwesome: для отображения иконок звездочек и других иконок.

## Примечание

Для использования приложения необходимо иметь доступ к интернету, так как приложение получает данные о фильмах через API The Movie Database (TMDb) и ещё неоходим VPN.
